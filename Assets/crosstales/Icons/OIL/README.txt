https://www.assetstore.unity3d.com/#!/content/98718?aid=1011lNGT



## Description
OIL (short for Open Icon Library) is an open-source library that consists of over 2’600 icons.
We created and exported images for every icon in various dimensions in the highest possible quality and prepared it to use in Unity as sprite or as texture.
